2.贴现现金流量分析一净现值法(DCF一NPV) DCF一NPV法是计算储量价格最常见的方法。通常是在油田勘查工作进行到一定程度,己经能够大致确定该油田的储量数量,并已了解其储层物性和油藏条件时,才采用该方法。DCF一NVP法是以一定的收益率作为贴现率,将想要购买的油气储量资产在其未来开发生产寿命期内历年的净收入贴现,求得净现值,并以此净现值作为储量价格的参考值。根据贴现的收益率的不同选择,这种方法又可具体分为目标内部收益率法和风险现值法。 (l)目标内部收益率法 该方法是以投资者所期望的目标内部收益率作为贴现率计算的未来从待评价的储量中获得的现金流的净现值。用该方法计算储量价值的公式为(根据我们分析,在计算净现值时,不考虑勘探投资。如果计算净现值时考虑勘探投资,则储量购买价格=勘探投资+NPV): V=∑_(t=1)^T▒〖(CIt-COt)*〗 〖(1+TIRR)〗^(-t) 其中,CIt—第t年的现金流入 COt—第t年的现金流出 TIRR—目标内部收益率,目前,通常取20% T—计算期,为可采储量的经济合理采出年限,包括净现金流(CIt一COt)大于0的所有年份。 在美国,石油工业投资通常寻求15%的税后收益率。美国投资者在国外寻求更高的税后收益率,通常达20%。该方法的优点是考虑了所有预测的生产因素以及包括通货膨胀在内的所有预测的经济因素,缺点是对风险缺乏灵活性。 (2)风险现值法` 用风险现值法计算储量市场价值的过程分为两步,第一步先用某一贴现率(美国证券和股票交易委员会规定采用10%的贴现率)贴现未来净现金流,得到净现值,再用风险因子调整净现值得到储量的价值。 即:NPV=∑_(t=1)^T▒〖(CIt-COt)/〖(1+i)〗^t 〗 V=NPV*R 其中,i—贴现率 R—风险系数或风险因子与其它方法相比,风险现值法的优点是它不但包括了所有的生产因素和经济因素,而且也考虑了风险。比如,对于生产寿命期很长的油田的储量资产,在计算其净现值时,如果用较高的贴现率计算,其净现值会很低,不能反映其真实价值。在美国,对于这种长寿油田的石油储量资产,常以比边际收益率低的贴现率(5一10%)求出净现值,然后乘以较高的风险系数(通常是67一75%),得出油气储量资产购买价格的参考值。
